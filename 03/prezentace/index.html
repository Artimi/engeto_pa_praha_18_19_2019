Conditions


				if variable:    ==      if bool(variable):
			

				bool(1) == True
				bool(0) == False
				bool('something') == True
				bool('') == False
				bool([]) == False
				bool([0]) == True

Logical operators - not


			not True == False
			not False == True

Logical operators - and


				True and True == True
				True and False == False
				False and True == False
				False and False == False

Logical operators - or


				True or True == True
				True or False == True
				False or True == True
				False or False == False

Logical operators - evaluation


				c1 and c2 and c3 and  c4
				c1 and (c2 and (c3 and c4)))
Short-circuit evaluation - if c1 is False other conditions are not evaluated. If you are not sure about precedence - use parentheses!

Logical ternary operator


			    half = height // 2 if height is not None else None

Today's lesson

Dictionary

  • condition - podmínka
  • integer - celé číslo
  • python - krajta
  • string - řetězec

						words = {
							'condition': 'podmínka',
							'integer': 'celé číslo',
							'python': 'krajta',
							'string': 'řetězec',
						}

Getting value


				>>> words = {
				...     'condition': 'podmínka',
				...     'integer': 'celé číslo',
				...     'python': 'krajta',
				...     'string': 'řetězec',
				... }
				>>> words['python']
				'krajta'

Getting value - missing key


				>>> words['java']
				Traceback (most recent call last):
				  File "<stdin>", line 1, in <module>
				KeyError: 'java'
				>>> words.get('java', 'MISSING')
				'MISSING'

Adding / updating


			>>> words['java'] = 'jáva'
			>>> words['python'] = 'hroznýš'
			>>> words
			{
				'condition': 'podmínka',
				'integer': 'celé číslo',
				'java': 'jáva',
				'python': 'hroznýš',
				'string': 'řetězec'
			}

Deleting


			>>> del words['java']
			>>> words.pop('python')
			'hroznýš'
			>>> words
			{
				'condition': 'podmínka',
				'integer': 'celé číslo',
				'string': 'řetězec'
			}

Iterating


				>>> words
				{
					'condition': 'podmínka',
					'integer': 'celé číslo',
					'string': 'řetězec'
				}
				>>> words.keys()
				dict_keys(['condition', 'integer', 'string'])
				>>> words.values()
				dict_values(['podmínka', 'celé číslo', 'řetězec'])
				>>> words.items()
				dict_items([('condition', 'podmínka'), ('integer', 'celé číslo'), ('string', 'řetězec')])

Nesting


				>>> words['python'] = {'nature': 'krajta', 'it': 'programovací jazyk'}
				>>> words
				{
					'condition': 'podmínka',
					'integer': 'celé číslo',
					'python': {'it': 'programovací jazyk', 'nature': 'krajta'},
					'string': 'řetězec'
				}
				>>> words['python']['it']
				'programovací jazyk'

Update with another dict


			>>> words.update({'float': 'desetinné číslo', 'string': 'řetěz'})
			>>> words
			{
				'condition': 'podmínka',
				'float': 'desetinné číslo',
				'integer': 'celé číslo',
				'python': {'it': 'programovací jazyk', 'nature': 'krajta'},
				'string': 'řetěz'
			}

Set


				>>> s = {1, 1, 2, 3, 8}
				>>> s
				{8, 1, 2, 3}
			    >>> 3 in s
			    True
				>>> set('Engeto academy')
				{'o', 'e', 'y', 't', 'c', 'm', ' ', 'a', 'E', 'd', 'n', 'g'}

Adding / removing


				>>> s.add(6)
				>>> s
				{1, 2, 3, 6, 8}
				>>> s.remove(3)
				>>> s
				{1, 2, 6, 8}
				>>> s.pop()
				1 # random element

Comparison


			    >>> s1 = {1, 2, 3}
			    >>> s2 = {2, 3}
			    >>> s1 <= s2  # s1.issubset(s2)
			    False
			    >>> s1 >= s2  # s1.issuperset(s2)
			    True

Set operations


				>>> s1 = {1, 2, 3}
				>>> s2 = {2, 3, 4}
				>>> s1 | s2  # s1.union(s2)
				{1, 2, 3, 4}
				>>> s1 & s2  # s1.intersection(s2)
				{2, 3}
				>>> s1 - s2  # s1.difference(s2)
				{1}
